# 1 背景介绍
    为了实现“数字校园建设覆盖全体学校”的目标，启动实施数字校园规范普及计划，4月16日教育部发布了《中小学数字校园建设规范（试行）》，为落实数字校园规范普及计划指明了方向。结合多年的行业知识，并根据国家相关规范，推出了拥有完全自主知识产权的数字化校园管理SaaS平台-超级云校。该平台能很好的将互联网技术和数据化运营的理念融入到学校信息化服务过程中，帮助学校客户实现转型和升级。
    2018年4月13日，教育部颁布《教育信息化2.0行动计划》，提出“三全两高一大”的发展目标，即“教师应用覆盖全体教师，学生应用覆盖全体适龄学生，数字校园建设覆盖全体学校”，“教师信息化应用水平普遍提高、师生信息素养普遍提高”，“建立互联网+教育大平台”，标志着我国教育信息化的发展核心目标从“三通两平台”转向“三全两高一大”。为了实现“数字校园建设覆盖全体学校”的目标，启动实施数字校园规范普及计划，4月16日教育部发布了《中小学数字校园建设规范（试行）》，为落实数字校园规范普及计划指明了方向。
    结合多年的行业知识，并根据国家相关规范，推出了拥有完全自主知识产权的数字校园管理SaaS平台-超级云校。该平台能很好的将互联网技术和数据化运营的理念融入到学校信息化服务过程中，帮助学校客户实现转型和升级。
    整体设计依据以下原则：
* 实用性：
  充分满足学校信息化应用与管理在现在及未来的各种需求，为学校服务与管理流程的优化与再造，提升了校园管理的效能提供强有力的支撑。
* 标准性：
  本平台的所有设计均遵循国家、行业、企业标准进行，以符合系统的开放性要求。
* 高可靠性：
  本平台设计通过合理而先进的设计及优化选项，保证系统的可靠性和容错性，避免事故发生。
* 高性能、先进性：
  平台全面采用微服务。通过CQRS（命令查询职责分离），前后端分离、动静资源分离、二级缓存等技术，有效的提高系统对大并发的支持。
* 可维护性：
  建立高内聚，低耦合模块。使用结构化程序设计技术，提高现有系统的可维护性。
